Logistics Management

This is a web application for a logistics firm in USA. It has the full functionality to manage both employees and business. Each employee in the company has a unique account in our web application they can manage their business, like add load, schedule load, notify the admin, request a load and reports.

Technology : Asp.Net MVC, Jquery and MS Sql

Hotel Booking

This system is mainly focusing on hotels and peoples in New Zealand. People who search for hotels can register in to our web application and send request to hotels. There is a real time chat functionality to discuss the rates with hotel agents.

Technology : Asp.Net API, AngularJS and MS Sql

Waste Management Reporting Portal

This is a reporting portal for a waste management company situated in USA. We used charts, calendar and tables to visualize the data and for charts we used AmChart.js. Users has the ability to export the reports as excel, pdf and image.

Technology : Asp.Net API, AngularJS and MS Sql
